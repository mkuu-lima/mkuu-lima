# Mkuu Lima

**The all-in-one farming app built for Ugandan farmers**

Buy seeds & fertilizer • Sell crops & livestock • Rent tractors • Manage your farm • Get expert advice

### Features
- Marketplace for buying inputs and selling produce
- Tractor rental & consultation services
- Farm planning tools
- Notifications & order tracking
- Built with Flutter (Android + iOS)
